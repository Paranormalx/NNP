ProjectF

Questo è un progetto fatto solo da una persona, quindi non ha molta qualità, però per farlo ci ho messo molto impegno.

Questo progetto consiste nel formare varie pagine HTML, con varie funzione, come l'orologio, o un drum kit.

Per realizzare questo progetto ho usato: HTML, Java Script e CSS.

Numeri casuali genera un numero casuale, tra 0 a 10, a 100 o a 1000, a scelta.

Drum kit, permette, premendo dei tasti, di simulare i suoni di una batteria.

Clock è un semplice orologio, con le lancette delle ore e dei minuti.
